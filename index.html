Content and data issues

Data of COVID-19 cases is retrieved from Johns Hopkins University github repository. Browse it to know more about potential errors or data issues.

Population data is from the World Bank for most of the countries and for national statistics institute. Population data is from 2018 and is stored here github.com/louisdecharson/covid-19/population_data.csv

The code is on Github.

Known data issues
